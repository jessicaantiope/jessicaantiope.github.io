Intro to Javascript

How does Javascript compare to html and css?

Html and Css make a website have content, structure and look good. Javascript uses makes a website interactive and can run more complicated behind the scene checks on data.

Explain control flow and loops

Control flow is when you use a statement to run a code based on certain conditions that you give it. Loops repeat codes over and over so you dont have to keep writing the same code.

Explain accessing data from arrays and objects

You can access all sorts of data from arrays as it is just a list made up of whatever fields you want to put into it. Each data type that you put in will have a numbered order to help you write different code using the array. Objects contain data and data functions which we can access using a key that we call a data type when entered.

Explain functions

Codeacademy refer to functions as being like a recipe which is a really good way to put it. When you write code in a certain way the functions will produce the desired outcome.
